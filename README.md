# 🎓 Student Toolbox

Student Toolbox is a simple web app that helps students manage their daily study tasks in one place.  
It includes a GPA calculator, Pomodoro timer, notes system, and text summarizer.

---

## 🚀 Features

### 📊 GPA Calculator
- Supports 4.0 and 4.3 grading scales
- Add multiple subjects
- Automatic GPA calculation

### ⏱️ Pomodoro Timer
- Focus and break sessions
- Custom work and break time
- Session tracking system

### 📝 Notes System
- Add and delete notes
- Search through notes
- Saves data using localStorage

### 📄 Text Summarizer
- Extracts important sentences from text
- Simple AI-like summarization (no API needed)
- Clean bullet-point output

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API
